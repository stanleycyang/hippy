#Hippy

[Hippy'](https://www.npmjs.com/package/hippy) application generator.

[![NPM Version][npm-image]][npm-url]
[![NPM Downloads][downloads-image]][downloads-url]
[![Linux Build][travis-image]][travis-url]

## Installation

```sh
$ npm install -g hippy
```

## Quick start

Create the app:

```bash
$ hippy demo_app
```

Install dependencies:

```bash
$ npm install
```

Start

```bash
$ hippy start
```
